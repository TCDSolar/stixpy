Added support for STIX energy-calibration (CAL) FITS files. `~stixpy.product.Product` now recognises ``cal-energy`` files via the new `~stixpy.product.sources.cal.EnergyCalibration` class. A `~stixpy.product.sources.cal.find_energy_calibration_file_for_time` helper applies the documented strategy (3-day window, on-board-ELUT match, ≥12 h recording duration, closest mid-time) to pick the daily calibration file applicable to a given instant or flare time range.
