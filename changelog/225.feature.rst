Add support for the STIX L3 flare list product via the new `~stixpy.product.sources.flarelist.FlareList` class. Flare list files can be discovered with `~stixpy.net.client.STIXClient` using the ``stix.DataType.flarelist`` attribute, loaded with `~stixpy.product.Product`, concatenated across files, filtered by boolean mask, and visualised with the bundled `~stixpy.product.sources.flarelist.FlareList.plot_timeseries` and `~stixpy.product.sources.flarelist.FlareList.plot_locations` helpers. Flare locations are exposed in both ICRS and Heliographic Stonyhurst frames, with location plots coloured by GOES class and projectable from Earth's or Solar Orbiter's point of view.
